1. (среднее) Реализовать переключение ссылок в верхней панели (Спорт / Экономика / Стартапы ...).

По умолчанию подсвечена ссылка "Спорт", нужно что бы при нажатии на "Экономику", подсвечивалась она, а "Спорт" гасла.

Подсказка:
Нужно убрать класс "active" у текущей кнопки, и добавить этот класс к новой кнопке,
с помощью методов addClass(), removeClass()

2. (сложное) Добавить к каждой новости кнопку "Не хочу это видеть", которая будет скрывать новость из списка

Подсказка:
Нужно использовать помощью методы parent() чтобы найти контейнер с новостью и метод hide() чтобы скрыть элемент
